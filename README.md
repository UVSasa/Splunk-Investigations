# 🔍 Splunk Security Investigations Lab

A curated collection of **realistic security investigations using Splunk**, designed to demonstrate my ability to think like a SOC analyst, investigate endpoint telemetry, and analyze adversary behavior using Windows Event Logs and Sysmon data.

---

## 🎯 Purpose

This repository showcases hands-on experience with:
- Security monitoring and detection engineering in Splunk
- Investigating endpoint telemetry (Sysmon + Windows Event Logs)
- Identifying suspicious behavior using real-world attack patterns
- Building structured SOC-style investigations from raw log data

The goal is to simulate how a SOC analyst would:
> Detect → Investigate → Correlate → Conclude

---
## 🧠 Skills Demonstrated

- Splunk Search Processing Language (SPL)
- Sysmon log analysis (Event IDs 1, 3, 10, 11, 22, etc.)
- Process tree reconstruction
- Network traffic investigation via logs
- Endpoint detection and response (EDR-style thinking)
- Threat hunting methodology
- MITRE ATT&CK mapping (where applicable)
